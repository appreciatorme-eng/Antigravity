# Servers

This directory contains full MCP Server implementations.

## n8n-mcp
Source: [https://github.com/czlonkowski/n8n-mcp](https://github.com/czlonkowski/n8n-mcp)

This is a complete MCP server that provides tools for interacting with n8n.
To use it, you generally need to run it (e.g. `npm install && npm start` inside the directory) and configure your MCP client to connect to it.
